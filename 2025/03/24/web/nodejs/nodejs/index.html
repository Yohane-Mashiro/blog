终极防御建议

  1. 最小权限原则：Node.js 进程以低权限用户运行。
  2. 输入过滤：禁止用户输入进入 exec、eval 等。
  3. 依赖检查：避免使用恶意第三方包（如 npm audit）。
  4. WAF：使用 Web 应用防火墙拦截可疑请求。
  5. 沙箱隔离：敏感操作放在受限环境（如 Docker、vm2）。
