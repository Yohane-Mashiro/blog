SSH 密钥配置

为了安全地连接 GitHub，需要配置 SSH 密钥：

  1. 生成 SSH 密钥对：

  2. 密钥默认生成在 ~/.ssh 目录下：

    • 公钥文件：id_rsa.pub
    • 私钥文件：id_rsa

  3. 将公钥添加到 GitHub：

    • 复制 id_rsa.pub 文件内容
    • 访问 https://github.com/settings/keys
    • 点击 “New SSH key” 并粘贴公钥内容
